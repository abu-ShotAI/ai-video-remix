# Composition 场景扩展规划

> 基于 ShotAI 镜头级语义理解 + Remotion 合成能力，可做到的视频混剪场景全览。
>
> 核心差异点：这些场景都**需要理解镜头内容**才能做好剪辑——知道这个镜头是特写还是广角、是激烈还是平静、情绪是什么、属于哪个叙事阶段——而不是简单按时间排列。

### 优先级建议

按 **商业价值 × 语义理解难度** 综合排序：

| 优先级 | Composition ID | 推荐理由 |
|--------|---------------|---------|
| **P0** | WeddingCinematic | C端高频刚需，情绪理解价值极高，付费意愿强 |
| **P0** | RetailBrandFilm | B端高频需求，品牌视频生产量大，流程标准化 |
| **P1** | CorporateStory | B端决策链短，预算充足，采访+B-roll理解典型场景 |
| **P1** | FitnessCoaching | C端大市场，动作分解语义理解场景具有代表性 |
| **P1** | ConcertLive | 演出行业，多机位语义理解价值突出 |
| **P2** | GamingMontage | 年轻用户，病毒传播潜力，游戏事件识别独特场景 |
| **P2** | FashionWeekHighlight | 时尚品牌预算高，走秀节奏+细节识别场景专业度高 |

## 各场景详细说明

### 1. WeddingCinematic — 婚礼电影感混剪

**适用人群**: 婚礼摄影摄像工作室、婚庆策划公司、个人留存

**素材来源**:
- 婚礼当天全程多机位素材（室内仪式 + 室外拍摄）
- 准备阶段（化妆、穿婚纱、新郎等待）
- 仪式片段（入场、宣誓、交换戒指、亲吻）
- 宴会、亲友反应、抛花束

**需要理解什么**:
- 仪式关键节点识别：入场 / 宣誓 / 亲吻 / 抛花束
- 人物情绪强度：哭泣、大笑、深情凝视、拥抱
- 新人特写 vs 亲友反应镜头的比例
- 慢镜素材的识别与情感高潮位置
- 场景类型区分：室内仪式 / 室外外景 / 宴会厅

**剪辑形式**:
- 情绪弧线：期待 → 高潮仪式 → 温情片刻 → 喜悦宴会
- 大量慢镜 + 特写，配合抒情配乐
- 色调：胶片感暖调 或 清冷梦幻，依风格而定
- 时长：3-5分钟精华版 / 10-15分钟完整版

---

### 2. RetailBrandFilm — 零售/电商品牌短片

**适用人群**: 消费品品牌、电商平台、DTC 品牌营销团队

**素材来源**:
- 产品拍摄（棚拍 + 生活场景）
- 用户真实使用场景录像
- 品牌故事 / 工厂溯源素材
- KOL 种草内容（授权）

**需要理解什么**:
- 产品特写细节 vs 使用场景 vs 情感共鸣镜头分类
- 产品核心卖点镜头识别（功能演示、质感、颜色）
- 用户状态：使用过程、满足感表情
- 场景定位匹配：高端 / 大众 / 户外
- 品牌色调元素识别

**剪辑形式**:
- 叙事结构："问题 → 产品出现 → 解决体验"
- 产品特写密集，节奏快但有设计感
- 色调与品牌 VI 统一
- 时长：15-30秒信息流广告版 / 60-90秒品牌片

---

### 3. CorporateStory — 企业品牌纪录片

**适用人群**: 企业公关/品宣部门、招聘团队、IR 投资者关系

**素材来源**:
- 办公室日常拍摄素材（工作、会议、协作）
- 产品/服务使用场景
- 员工采访、CEO 陈述片段
- 客户现场使用镜头

**需要理解什么**:
- 采访镜头（面对镜头说话）vs 工作场景镜头（自然状态）的分类
- 室内 / 室外、正式 / 非正式场景识别
- 人物情绪：专注、愉悦、协作
- 产品/服务使用的过程动作细节
- 公司 Logo、标志性空间识别

**剪辑形式**:
- 采访音频为主线，B-roll 填充叙事
- 低饱和专业色调，强调真实感
- 节奏舒缓，有呼吸感，非快剪
- 时长：2-4分钟

---

### 4. FitnessCoaching — 健身/运动教学

**适用人群**: 健身博主、运动品牌、在线健身平台（Keep 等）

**素材来源**:
- 动作示范录像（多角度拍摄）
- 训练现场记录
- 教练讲解素材
- 学员前后对比

**需要理解什么**:
- 动作阶段识别：准备姿势 / 发力 / 还原
- 角度类型：正面（对称性）/ 侧面（动作幅度）/ 特写（关节位置）
- 运动强度：慢速示范 vs 正常速度 vs 高强度爆发
- 正确动作 vs 错误动作对比识别
- 教练 vs 普通学员的画面区分

**剪辑形式**:
- "示范 → 讲解 → 实战" 三段循环结构
- 关键动作慢镜 + 标注字幕
- 高强度训练段快剪配热血音乐
- 时长：8-15分钟教学向 / 30-60秒推广片

---

### 5. ConcertLive — 演唱会/Live 现场集锦

**适用人群**: 音乐厂牌、演出主办方、音乐人团队

**素材来源**:
- 演唱会多机位导播录像
- 舞台侧面花絮素材
- 观众区拍摄（UGC 整合）
- 后台准备素材

**需要理解什么**:
- 表演高潮段落识别（高音时刻、间奏舞蹈）
- 观众反应：举灯、跟唱、情绪爆发
- 舞台灯光状态：追光、色彩变换、烟雾效果
- 表演者肢体动作强度
- 多机位景别调度：全景 / 中景 / 演员特写 / 观众

**剪辑形式**:
- 严格踩音乐 beat 剪辑
- 副歌/高潮密集切换，verse 段落舒缓
- 暗调高对比，灯光色彩保留
- 时长：单曲 MV 剪辑版 3-4分钟 / 精华集锦 5-8分钟

---

### 6. GamingMontage — 游戏精彩集锦

**适用人群**: 游戏主播、电竞战队、游戏发行商

**素材来源**:
- 游戏录屏（Screen Capture）
- 赛事转播素材
- 主播面部摄像头（Facecam）
- 观众/队友反应

**需要理解什么**:
- 游戏内高光事件识别（击杀、翻盘、绝杀）
- 玩家面部情绪：紧张、爆笑、崩溃、庆祝
- 游戏画面节奏：平静探索 vs 战斗爆发
- 多视角同步：玩家视角 / 选手面部 / 观众席
- 比赛关键节点：开局 / 僵局 / 决胜时刻

**剪辑形式**:
- 极速快剪 + 音效强化
- 高光时刻慢镜 + 放大 + 音效爆炸
- 暗调高饱和，电竞视觉风格
- BGM 以 EDM / Trap 为主踩点
- 时长：2-4分钟

---

### 7. FashionWeekHighlight — 时尚发布会集锦

**适用人群**: 时尚品牌市场团队、造型师工作室、时尚媒体

**素材来源**:
- 品牌秀场多机位录像
- 后台花絮、妆造记录
- 买手/观众反应镜头

**需要理解什么**:
- 全身走秀镜头 vs 局部服装细节特写
- 模特步态节奏（踩 beat 剪辑依赖）
- 面部表情与情绪（自信、冷峻、愉悦）
- 布料质感、印花、配饰近景识别
- 观众席鼓掌 / 惊叹反应镜头

**剪辑形式**:
- 节奏跟音乐 beat 强剪：全身 → 特写 → 反应三段循环
- 高对比色调或品牌色调覆盖
- 结尾慢镜定格标志性造型
- 时长：60-90秒传播版 / 3-5分钟完整回顾版

---

### 8. FoodTasting — 美食探店/料理节目

**适用人群**: 美食博主、餐厅品牌、烹饪平台

**素材来源**:
- 餐厅探访录像
- 厨房制作过程（翻炒、摆盘）
- 食物出餐镜头
- 博主试吃反应

**需要理解什么**:
- 食物特写（有食欲感）vs 环境氛围镜头
- 制作过程关键步骤（翻炒、摆盘、淋汁）
- 试吃表情反应（满足、惊喜）
- 餐厅环境风格（高档 / 街边 / 家常）
- 食物状态：热气、流心、层次感

**剪辑形式**:
- 快节奏特写驱动，满足感官刺激
- "烹饪过程 → 成品展示 → 试吃反应" 三段式
- 暖色调，饱和度略高
- 时长：60秒短视频版 / 5-8分钟完整版

---

### 9. ArchitectureTour — 建筑空间导览

**适用人群**: 建筑事务所、地产开发商、室内设计品牌、文旅机构

**素材来源**:
- 无人机航拍（外观/全景）
- 室内推轨/稳定器拍摄
- 建造过程延时摄影
- 设计师讲解素材

**需要理解什么**:
- 外观全景 vs 室内空间 vs 材质细节的分类
- 相机运动类型：推进 / 环绕 / 升降（与叙事匹配）
- 光线状态：自然光 / 人工光 / 黄金时段
- 人的存在感：有人活动的空间 vs 纯空间展示
- 建筑材质特写：木、石、玻璃、金属

**剪辑形式**:
- 空间叙事结构：外部 → 入口 → 核心空间 → 细节 → 人的使用
- 慢节奏，长镜头为主
- Ambient / Minimal 配乐
- 时长：90秒推广版 / 4-6分钟完整导览版

### 10. StartupDocumentary — 创业纪录片

**适用人群**: 创业公司品宣、投资机构（被投企业故事）、商业媒体

**素材来源**:
- 创始人日常工作记录
- 团队会议、头脑风暴
- 产品发布/客户演示现场
- 创始人专访

**需要理解什么**:
- 采访镜头 vs 工作状态镜头分类
- 压力状态 vs 突破时刻的情绪转变识别
- 场景区分：办公室 / 咖啡馆 / 会议室 / 发布会现场
- 产品屏幕 / 代码 / 设计稿等工作内容镜头
- 早期简陋环境 vs 成长后规模的时间线线索

**剪辑形式**:
- 叙事结构：问题 → 挣扎 → 转折 → 成果
- 采访音频为叙事主轴，B-roll 填充
- 真实感优先，不过度美化
- 时长：3-6分钟

### 11. UrbanStreet — 城市街头纪实

**适用人群**: 街头摄影师、城市文化媒体、旅游局城市宣传

**素材来源**:
- 街头抓拍素材（手持、随拍风格）
- 城市不同时段（清晨 / 夜晚 / 雨天）
- 市场、地铁、街角等典型场景

**需要理解什么**:
- 人物状态：匆忙通行 vs 停留驻望 vs 互动中
- 时间段识别：光线判断（蓝调时刻 / 正午 / 夜晚霓虹）
- 城市元素：建筑标志、交通工具、广告牌语言
- 镜头感：长焦压缩感 vs 广角环境感
- 氛围元素：雨、雾、光线折射

**剪辑形式**:
- 意识流结构，非线性叙事
- 节奏跟随配乐情绪
- 高对比黑白 或 脱饱和城市色调
- 时长：2-4分钟

### 12. TravelShortFilm — 深度旅行短片（叙事向）

> 区别于已有的 **TravelVlog**（日记流水账），这里强调**叙事性**和**文化深度**

**适用人群**: 旅行内容创作者（中高端）、旅游局、纪录片团队

**素材来源**:
- 目的地多景别精心拍摄素材
- 当地人生活场景
- 文化仪式 / 节庆活动
- 旅行者独白 / 采访

**需要理解什么**:
- 地域文化符号识别：建筑风格、服饰、饮食
- 叙事节点镜头：初见 / 迷路 / 相遇 / 离别
- 当地人与旅行者的互动状态
- 景别叙事层次：远景（建立地点）→ 中景（人物关系）→ 近景（情感细节）
- 时光感镜头：日出日落、人群流动、市场开关

**剪辑形式**:
- 有旁白或独白的叙事结构（区别于纯视觉 Vlog）
- 节奏克制，有沉浸感
- 胶片色调或低饱和写实风
- 时长：5-10分钟

---

### 13. MedicalEducation — 医疗培训视频

**适用人群**: 医院培训科、医疗器械公司、医学院校

**素材来源**:
- 手术/操作过程录像
- 模拟人操作训练素材
- 医护讲解录像
- 医疗设备操作流程视频

**需要理解什么**:
- 操作步骤阶段识别：准备 / 操作 / 确认
- 器械特写 vs 整体操作视角的切换时机
- 医护人员手部动作细节
- 关键操作时刻的定位（需要暂停/慢镜强调处）
- 错误操作 vs 正确操作对比场景

**剪辑形式**:
- 步骤编号字幕 + 关键操作慢镜重放
- 关键步骤停顿 + 语音讲解同步
- 对比剪辑（正确/错误）用分屏或顺序呈现
- 严谨清晰优先，不追求视觉炫技
- 时长：依流程定，通常 5-15分钟

### 14. IndustrialProcess — 工业/制造流程展示

**适用人群**: 制造企业品宣、工业展会、招商引资、B2B 营销

**素材来源**:
- 工厂生产线录像
- 机械设备操作特写
- 质检/测试过程
- 大型设备组装延时摄影

**需要理解什么**:
- 工艺阶段识别：原料 → 加工 → 组装 → 检测 → 成品
- 机械动作精度镜头（焊接、切割、精密装配）
- 规模感镜头：工厂全景 vs 局部精细操作
- 人机协作状态与熟练度体现
- 产品变化过程：零件 → 成品的视觉演化

**剪辑形式**:
- 流程线性叙事，逻辑清晰优先
- 延时摄影展示大尺度变化
- 科技感、稳健配乐
- 工艺参数/质检标准字幕（可选）
- 时长：2-4分钟

### 15. ChildhoodMemory — 家庭/童年回忆剪辑

**适用人群**: 个人用户、家庭相册应用、周年纪念视频服务

**素材来源**:
- 家庭手机录像（多年跨度）
- 老照片翻拍视频
- 节日/生日/旅行记录

**需要理解什么**:
- 儿童年龄段大致识别（婴儿 / 幼儿 / 儿童 / 青少年）
- 情感时刻识别：第一次站立、生日许愿、家人拥抱
- 节日场景识别：春节 / 圣诞 / 生日等符号元素
- 画质年代感：老旧素材 vs 现代素材用于时间线排序
- 家庭成员关系：亲子互动、兄弟姐妹、祖孙代际

**剪辑形式**:
